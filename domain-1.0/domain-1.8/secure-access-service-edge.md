
# Secure Access Service Edge (SASE)

Allows secure access for cloud services from different locations. SASE can be though of as a next generation VPN. Our security technologies are in the cloud and SASE clients would be on all devices.

Below are a few examples of what SASE has to offer.

## Network As A Service
- SD-WAN
- VPNs
- QoS
- Routing
- SaaS acceleration

## Security as a Service
- Zero Trust Network Access
- Cloud secure web gateway
- Cloud access security broker
- Firewall as a service
- DLP
- DNS Security
- Threat Prevention

## User Experience
- SASE Clients on all devices
- Automatic Connections
- Consistent Processes
